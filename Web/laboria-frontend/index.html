Sistema de análisis predictivo y visualización avanzada del mercado laboral de la Comunidad Valenciana.

Sobre el Proyecto

LaborIA no es solo un cuadro de mandos, es una infraestructura completa de Big Data diseñada para entender el pasado y predecir el futuro del empleo local.

El Reto (Big Data)

El Servicio Público de Empleo Estatal genera volúmenes masivos de datos. El reto principal ha consistido en limpiar, estructurar e indexar más de 22.5 millones de registros históricos, diseñando una caché hiper-optimizada en MongoDB capaz de responder al navegador en menos de 50 milisegundos.

Inteligencia Artificial

Cruzando variables sociodemográficas y sectoriales, hemos entrenado un modelo de Machine Learning (Random Forest) encapsulado en una API de Python (Flask). Este agente actúa como orientador laboral, identificando patrones ocultos para recomendar el municipio con mayor probabilidad de contratación.

Arquitectura Cloud

Desplegado íntegramente en la nube mediante un clúster en Amazon Web Services (AWS). Cuenta con un proxy inverso Nginx y encriptación SSL (Let's Encrypt), separando el frontend perimetral en Cloudflare del procesamiento pesado backend (Node.js) mediante una arquitectura de microservicios segura.
